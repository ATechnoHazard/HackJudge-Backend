## Instructions to run

* Pre-requisites:
	- Node.js
	- NPM

* Setting up the app
```bash
git clone https://github.com/GDGVIT/HackJudge-Backend
cd HackJudge-Backend
```

* Setting up the environment
	- Set the environment variable ```MONGO_ATLAS_PW``` to your mongodb instance

* Running the app

```bash
npm install
npm start
```
